# 💳 UPI Fraud Detection & Awareness System ⚠️

## 📌 Project Overview
This project focuses on detecting and preventing UPI-based frauds by analyzing suspicious transaction patterns and spreading digital awareness among users. It is designed especially for students and beginners who are new to digital payments.

---

## ❗ Problem Statement
With the rapid growth of UPI transactions in India, cases of fraud such as fake collect requests, phishing links, and QR scams have increased. Many users fall victim due to lack of awareness and understanding.

---

## 💡 Proposed Solution
This system identifies potentially fraudulent transactions using rule-based logic and alerts users before they take risky actions. It also provides awareness tips to promote safe online behavior.

---

## 🎯 Objectives
- Detect common UPI fraud patterns  
- Alert users about suspicious transactions  
- Promote digital literacy and safe practices  
- Apply AI/ML concepts to a real-world problem  

---

## ⚙️ Features
- 🔍 Fraud Detection System (rule-based logic)  
- ⚠️ Real-time Warning Messages  
- 📚 Awareness Module with Safety Tips  
- 💻 Simple and User-Friendly Interface  

---

## 🛠️ Tech Stack
- Python  
- Basic Machine Learning Concepts (optional)  

---

## 🧠 How It Works
1. User inputs transaction details  
2. System checks conditions like:
   - Unknown sender  
   - Urgent payment request  
   - Suspicious QR usage  
3. If conditions match → ⚠️ Warning displayed  
4. Else → ✅ Safe transaction
